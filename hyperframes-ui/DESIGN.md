# HyperFrames UI Motion Design

## Style Prompt

Animate the TCM knowledge graph interface as a living herbarium: rice paper base, ink-green panels, pale jade relation nodes, cinnabar emphasis, and thin botanical linework. The motion should feel precise and medicinal rather than flashy: graph links pulse, labels reveal like ink strokes, and search-to-graph transitions bloom outward from the queried formula or herb.

## Colors

- Background: `#F4EFE3`
- Deep surface: `#143D2B`
- Text: `#1D2520`
- Leaf accent: `#4E8F5A`
- Jade panel: `#C8D8B8`
- Cinnabar: `#B9472E`
- Fiber line: `#E5D7BB`

## Typography

- Headline: high-contrast serif if available, fallback Georgia.
- UI/body: `Microsoft YaHei`, `PingFang SC`, sans-serif.
- Data labels: monospace fallback with tabular numbers.

## Motion Rules

- Build final layout first; animate elements into that layout.
- Use entrance animations on every scene.
- Use transitions between scenes, never jump cuts.
- Use finite ambient loops only.
- No infinite repeat timelines.
- No text overlap at desktop or mobile aspect ratios.

## What Not To Do

- No neon tech gradients.
- No bokeh or floating orb decoration.
- No dark-blue-only palette.
- No forced line breaks in long text.
